개인 프로젝트 구상

1안
기억력 게임)

중심을 기준으로 3 * 3 카드 만들기
시작하고 3초 뒤에 카드 2초간 공개 후 다시 숨기기
카드를 클릭하면 공개되었던 카드의 숫자 출력
숫자를 오름차순으로 나열하면 클리어, 중간에 틀리면 실패
-> 마지막엔 clear/fail 문구와 함께 점수 출력

+ 점수가 어느정도 오르면 난이도 높이기
랜덤으로 나오는 숫자의 범위 넓힘
1라운드 : 1 ~ 10
2라운드 : 1 ~ 20
3라운드 : 1 ~ 50
더 이상은 없음

2안
사칙연산 게임)

화면에 랜덤으로 사칙연산 문제를 주고 제한 시간을 둠
정답을 주관식 혹은 객관식으로 받아서 정답/오답 판별
총 3번 틀리면 게임오버, 게임오버시 Game Over 문구와 함께 점수 출력
생명은 처음에 3개 -> 틀리면 하나씩 없어짐(fade 효과)

+ 1안과 비슷하게 점수가 어느정도 오르면 난이도를 높여도 괜찮을듯(처음엔 덧셈 뺄셈만, 나중에는 복합연산이나 곱셈, 나눗셈 추가)

3안
레이튼 따라하기(수수께끼 몇문제 따와서 퀴즈 풀기, 답은 무조건 객관식으로)
-> 문제 실패하면 view 바꿔서 실패 이미지/대사 후 다시하기/포기하기 버튼 넣기(포기하면 다음 문제로 넘어감)
-> 성공하면 view 반짝캐럿 점수(점수 올림)/다음문제 버튼 넣기, 다음문제로 넘어감
-> 마지막 문제까지 가면 타이틀 화면으로 돌아옴
